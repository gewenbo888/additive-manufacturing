# Additive Manufacturing Engine · 增材制造引擎

A bilingual atlas of additive manufacturing — FDM, SLA, SLS, metal AM, bioprinting, construction-scale concrete printing, generative design, distributed fabrication, nanotechnology and space manufacturing — and the question of whether matter is becoming a programmable function of information.

Ten systems · a layer-by-layer build simulator · a manufacturing-eras capability ladder (stone → bronze → factory → CNC → robot → printer → bioprinter → construction → nano) · a materials lab tour · a six-axis paradigms radar (craft / mass / CNC / additive / distributed / self-replicating) · an L0→L5 AI generative-design ladder · a six-stage info-to-matter merge · a ten-layer recursive fabrication engine (raw feedstock → single machine → print farm → fab hub → planetary network → bio-integrated → atomically precise → orbital → interplanetary) · a six-lens AI Fabrication Analyst.
